Interactive 3D via eye-tracking using JavaScript and WebGL

This is a proof-of-concept that I put together in a few hours to examine the possibility of eyetracking-based 3D scene faking.
You have to allow this webpage to access your webcamera, otherwise this application won't work.
This app uses http://github.com/liuliu/ccv for face detection and WebGL for scene rendering. Example scene based on http://learningwebgl.com/blog/?p=239
